# Hello Reack Backend

> A simple Rails API backend.

## Built With

- Ruby
- Ruby on Rails
- Postgres

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Ruby 3.1, Ruby on Rails 7.0

### Setup

- Run `cd hello-rails-react-2-apps/hello-rails-back-end`
- Run `bundle install`
- Run `rails db:create && rails db:migrate && rails db:seed`

### Usage

- Run `rails s -p 4000`
